<h2>HisseTakip</h2>

Bu projede, Borsa İstanbul’da işlem gören hisselerin takibini ve analizini
kolaylaştırmayı amaçlayan web tabanlı bir uygulama geliştirilmiştir. Projenin temel
amacı, bireysel yatırımcıların farklı kaynaklardan edinilmesi gereken finansal verileri
tek bir platform üzerinden sade ve anlaşılır bir şekilde görüntüleyebilmesini
sağlamaktır. Uygulama, yatırım tavsiyesi sunmayı hedeflememekte; teknik
göstergeler, fiyat grafikleri ve kurumsal analiz özetleri aracılığıyla kullanıcıların kendi
karar süreçlerini desteklemeyi amaçlamaktadır.


<h2>Installation Steps:</h2>

<p>1. Install dependencies:</p>

```
npm install
```

<p>2. Generate prisma client</p>

```
npx prisma generate
```

<p>3. Start the project</p>

```
npm run dev
```
